# Python Code Challenges

This repository features all three code challenges implemented for the Advanced Python Programming course of my master's program.

Each code challenge was implemented in .ipynb format (Jupyter Notebook) for more detailed representation of the results, as well as a Python file (.py) for easier debugging.

## Code Challenge 1: Longest Increasing Path

Given a matrix of integers of mxn dimensions, the task is to find the length of the longest increasing path within the matrix.

## Code Challenge 2: Rock, Paper, Scissors variant

Given two players `(Player 1, Player 2)` over N rounds, both players throw either **Rock (R)**, **Paper (P)** or **Scissors (S)** 
at the same time during each round.

## Code Challenge 3: Random Dates Generator

Given a number of days passed since the establishment of a civilization, each date is represented through the utilization of integer values.
Therefore, self-describing dates are being generated.
